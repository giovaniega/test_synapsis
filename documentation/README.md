## Membuat Environment

```bash
python -m venv myenv
```

## Mengaktifkan Environment

(Windows)
```bash
myenv\Scripts\activate
```

(Linux)
```bash
source myenv/bin/activate
```

## Install Dependencies

```bash
pip install -r log/requirements.txt
```

## Menjalankan Program

Pastikan berada di direktori /soal_python, lalu jalankan:

```bash
python -m soal1.main
```
